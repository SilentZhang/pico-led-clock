# RP2350 LED 视觉时钟 - TinyUSB + LwIP + HTTP Server 版本

## 项目说明

这是一个完全实现 **TinyUSB + LwIP + HTTP Server** 的LED视觉时钟固件！

通过 USB 模拟网络设备（RNDIS），连接电脑后可以通过浏览器界面设置时间，无需额外脚本或服务！

## 文件结构

| 文件 | 说明 |

## 使用步骤

### 1. 编译固件

使用 Pico SDK 或 VS Code 编译项目：
```bash
mkdir build
cd build
cmake ..
make
```

### 2. 烧录到 RP2350

将生成的 `led_visual_clock.uf2` 烧录到 RP2350 开发板。

### 3. 配置 USB 网络设备静态 IP

将 RP2350 通过 USB 连接到电脑后，Windows 会识别为一个新的 USB 网络设备。

需要配置静态 IP：
1. 打开「网络连接」
2. 右键点击 RP2350 的 USB 网络设备
3. 选择「属性」→「Internet 协议版本 4 (TCP/IPv4)」
4. 选择「使用下面的 IP 地址」，填入：
   - IP 地址：`192.168.7.2`
   - 子网掩码：`255.255.255.0`
5. 点击「确定」

**⚠️ 重要：不要使用网络共享！这会让 WiFi 断网！**

### 4. 通过浏览器设置时间

打开浏览器，访问：`http://192.168.7.1`

网页会自动填充当前系统时间，点击「Set Time」按钮即可！

## LED 显示说明

| LED 颜色/模式 | 含义 |
|--------------|------|
| 红→绿→蓝 循环闪烁 | 启动中 |
| 红色闪烁 | 显示小时 |
| 绿色闪烁 | 显示刻钟（15分钟为1单位） |
| 蓝色闪烁 | 显示分钟余数 |
| 绿色闪烁2次 | 时间设置成功 |

## 时间显示规则

1. **小时**：红色 LED 闪烁，24小时制转换为12小时制
2. **刻钟**：绿色 LED 闪烁，每15分钟为1单位（0-3次）
3. **分钟**：蓝色 LED 闪烁，显示余数分钟（0-14次）

每段显示之间有 1.5 秒全灭间隔。
