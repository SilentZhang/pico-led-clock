# RP2350 LED 视觉时钟 - TinyUSB + LwIP + NTP 版本

## 项目说明

这是一个完全实现 **TinyUSB + LwIP + NTP** 的LED视觉时钟固件！

通过 USB 模拟网络设备（RNDIS），连接电脑后自动获取网络时间，无需任何手动操作！

## 文件结构

| 文件 | 说明 |
|------|------|
| `main.c` | LED时钟主程序 |
| `net.c` | TinyUSB+LwIP+NTP网络模块 |
| `led_clock.h` | 头文件和配置 |
| `tusb_config.h` | TinyUSB配置 |

| `ws2812.pio` | WS2812 LED驱动 |
| `CMakeLists.txt` | 项目构建配置 |
| `README.md` | 使用说明 |

## 使用步骤

### 1. 编译固件

使用 Pico SDK 或 VS Code 编译项目：
```bash
mkdir build
cd build
cmake ..
make
```

### 2. 烧录到 RP2350

将生成的 `led_visual_clock.uf2` 烧录到 RP2350 开发板。

### 3. 配置 Windows 网络共享

将 RP2350 通过 USB 连接到电脑后，Windows 会识别为一个新的 USB 网络设备。

需要配置网络共享：
1. 打开「网络连接」
2. 右键点击有互联网连接的网络适配器（如 Wi-Fi）
3. 选择「属性」→「共享」
4. 勾选「允许其他网络用户通过此计算机的 Internet 连接来连接」
5. 在下拉菜单中选择 RP2350 的 USB 网络设备

### 4. 自动获取时间

配置好网络共享后，RP2350 会自动：
- 通过 DHCP 获取 IP 地址
- 查询 DNS 获取 NTP 服务器地址
- 发送 NTP 请求获取准确时间
- 同步后闪烁绿色 LED 确认
- 每小时自动同步更新

## LED 显示说明

| LED 颜色/模式 | 含义 |
|--------------|------|
| 红→绿→蓝 循环闪烁 | 启动中 |
| 红色闪烁 | 显示小时 |
| 绿色闪烁 | 显示刻钟（15分钟为1单位） |
| 蓝色闪烁 | 显示分钟余数 |
| 绿色闪烁2次 | 时间同步成功 |

## 时间显示规则

1. **小时**：红色 LED 闪烁，24小时制转换为12小时制
2. **刻钟**：绿色 LED 闪烁，每15分钟为1单位（0-3次）
3. **分钟**：蓝色 LED 闪烁，显示余数分钟（0-14次）

每段显示之间有 1.5 秒全灭间隔。

## 配置说明

### NTP 服务器

修改 `net.c` 中的 `NTP_SERVER` 可以更换 NTP 服务器：
```c
#define NTP_SERVER "pool.ntp.org"  // 全球 NTP 池
// 或者
#define NTP_SERVER "cn.ntp.org.cn" // 中国 NTP 服务器
```

### 时区

修改 `net.c` 中的时区偏移：
```c
#define TIMEZONE_OFFSET (8 * 3600)  // UTC+8 北京时区
```

### 更新间隔

修改 `net.c` 中的更新间隔：
```c
if (ntp_received && (now - last_ntp_update) > 3600000) {  // 3600000ms = 1小时
```

## 注意事项

1. RP2350-one 没有硬件 Wi-Fi 模块，本方案通过 USB 共享网络
2. 需要在 Windows 上配置网络共享才能让 RP2350 访问互联网
3. 网络配置后，第一次同步可能需要数秒
4. 同步成功后会闪烁绿色 LED 确认
5. 每小时自动同步一次，保持时间准确
