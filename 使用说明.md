
# LED 时钟 - 使用说明

## 🎉 完美方案！

- ✅ **完整 TinyUSB + LwIP 网络方案**
- ✅ **完全不需要电脑脚本或服务！**
- ✅ **不影响 WiFi！**
- ✅ **只需要浏览器！**

---

## 📝 使用步骤

### 步骤 1: 烧录固件

在 VS Code 中编译并烧录固件到 RP2350 设备。

### 步骤 2: 设置静态 IP

1. 把 RP2350 插到电脑上
2. 打开：控制面板 → 网络和 Internet → 网络和共享中心
3. 左侧点击："更改适配器设置"
4. 找到新的网络设备（USB RNDIS Device）
5. 右键 → 属性 → Internet 协议版本 4 (TCP/IPv4)
6. 选择"使用下面的 IP 地址"，填入：
   - IP 地址：`192.168.7.2`
   - 子网掩码：`255.255.255.0`
7. 点击"确定"

**⚠️ 重要：不要使用网络共享！这会让 WiFi 断网！**

### 步骤 3: 打开网页设置时间

1. 打开浏览器
2. 访问：`http://192.168.7.1`
3. 输入当前时间（小时和分钟）
4. 点击"设置时间"
5. 完成！LED 会显示绿色表示成功！

---

## 📌 LED 状态指示

- 红色闪烁 = 无连接
- 蓝色 = 网络连接
- 紫色 = 等待时间设置
- 绿色 = 时间设置成功！
