
# LED 时钟 - 使用说明

## 🎉 完美方案！

- ✅ **完整 TinyUSB + LwIP 网络方案**
- ✅ **完全不需要电脑脚本或服务！**
- ✅ **不影响 WiFi！**
- ✅ **只需要浏览器！**
- ✅ **支持设置秒数！**
- ✅ **自动获取系统当前时间！**

---

## 📝 使用步骤

### 步骤 1: 烧录固件

在 VS Code 中编译并烧录固件到 RP2350 设备。

### 步骤 2: 设置静态 IP

1. 把 RP2350 插到电脑上
2. 打开：控制面板 → 网络和 Internet → 网络和共享中心
3. 左侧点击："更改适配器设置"
4. 找到新的网络设备（USB RNDIS Device）
5. 右键 → 属性 → Internet 协议版本 4 (TCP/IPv4)
6. 选择"使用下面的 IP 地址"，填入：
   - IP 地址：`192.168.7.2`
   - 子网掩码：`255.255.255.0`
7. 点击"确定"

**⚠️ 重要：不要使用网络共享！这会让 WiFi 断网！**

### 步骤 3: 打开网页设置时间

1. 打开浏览器
2. 访问：`http://192.168.7.1`
3. **网页会自动填充当前系统时间（小时、分钟、秒）！**
4. 如需调整，修改输入框中的值
5. 点击「Set Time」按钮
6. 完成！LED 会闪烁绿色表示成功！

## 📌 LED 状态指示

| LED 颜色/模式 | 含义 |
|--------------|------|
| 红→绿→蓝 循环闪烁 | 启动中 |
| 蓝色闪烁（网络连接） | 等待时间设置 |
| 绿色闪烁2次 | 时间设置成功！ |
| 红色闪烁 | 显示小时 |
| 绿色闪烁 | 显示刻钟 |
| 蓝色闪烁 | 显示分钟 |

---

## 🎯 时间显示规则

1. **小时**：红色 LED 闪烁，24小时制转换为12小时制
2. **刻钟**：绿色 LED 闪烁，每15分钟为1单位（0-3次）
3. **分钟**：蓝色 LED 闪烁，显示余数分钟（0-14次）

每段显示之间有 1.5 秒全灭间隔。

## 💡 常见问题

**Q: 网页加载不完全怎么办？**
A: 确保静态 IP 设置正确，然后刷新浏览器（Ctrl+F5）。

**Q: LED 不显示时间？**
A: 检查是否已经点击「Set Time」按钮设置了时间。

**Q: 时间不准确怎么办？**
A: 重新打开网页，刷新后会获取新的系统当前时间，再点击设置。

**Q: 这个方案影响 WiFi 吗？**
A: 完全不影响！我们只设置静态 IP，不需要网络共享。
